# 🏦 Transworld Home — Bank Payment Guide Widget

An interactive, fully self-contained **single-file HTML widget** that provides 
step-by-step 1BILL / 1LINK payment instructions for 21 Pakistani banks.

## 🚀 Features

- 🔍 **Live Search** — Filter banks instantly by name
- 📋 **Bank-Specific Steps** — Each of the 19 banks has its own unique payment flow
- 🎨 **Animated UI** — Smooth fade-in and slide transitions
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- ⚡ **Zero Dependencies** — No frameworks, no npm, no build step required

## 🛠️ Technologies Used

| Technology | Purpose |
|---|---|
| **HTML5** | Semantic page structure |
| **Vanilla CSS3** | All styling — CSS custom properties (variables), Flexbox, CSS Grid, `@keyframes` animations, media queries |
| **Vanilla JavaScript (ES5)** | DOM manipulation, event handling, dynamic rendering, bank steps lookup |
| **SVG** | All icons — inline, no icon library needed |
| **Google Fonts** | Typography — `Syne` (headings) + `Inter` (body) |
| **CSS Custom Properties** | Design token system (`--brand`, `--surface`, `--ink`, etc.) |
